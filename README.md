1. ĐĂNG NHẬP / ĐĂNG KÝ
Click "Đăng Nhập" trên menu

Tài khoản mẫu:

Tên: Cửu U Chân Nhân

Mật khẩu: password123

Hoặc đăng ký tài khoản mới

2. CHỨC NĂNG CHÍNH
Trang chủ: Xem thống kê, nút bắt đầu viết

Thư viện: Xem danh sách truyện, lọc thể loại

Viết truyện: Đăng truyện mới, lưu nháp

Động phủ: Xem profile, lịch sử, cài đặt tài khoản

Về chúng tôi: Thông tin nhóm phát triển

3. ĐỌC TRUYỆN
Vào Thư viện

Click "Đọc" trên truyện muốn xem

Có thể:

Thả tim (Like)

Bình luận

Xem nội dung đầy đủ

4. HỆ THỐNG KINH NGHIỆM
Mỗi truyện đọc được = +50 XP

Tăng cấp → Thay đổi cảnh giới tu luyện

Xem XP ở Động phủ

5. LỊCH SỬ TU LUYỆN
Vào Động phủ → tab Lịch Sử

Xem:

Truyện đã viết

Truyện đã like

Bình luận đã gửi

6. SỬA THÔNG TIN
Vào Động phủ → tab Cài Đặt

Đổi biệt danh / mật khẩu

LƯU Ý NHANH:
Dữ liệu lưu trong trình duyệt (LocalStorage)

Refresh trang vẫn giữ đăng nhập

Video nền tự động chạy

Menu mobile: click icon ☰
